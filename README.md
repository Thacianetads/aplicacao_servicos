# 🛠️ Sistema de Gestão de Oficina Mecânica

Este projeto foi desenvolvido como desafio de conclusão de treinamento. O objetivo do sistema é registrar e gerenciar os serviços de uma Oficina Mecânica, fornecendo dados essenciais para o atendimento ao cliente e inteligência de negócio através de relatórios de custos, lucros e resumos financeiros.

---

## 📐 Arquitetura do Sistema e Camadas

O sistema foi planejado utilizando uma arquitetura em camadas bem definidas para garantir a separação de responsabilidades. Abaixo estão detalhadas as diretrizes de design, com foco especial nas camadas de **Persistência** e **Negócio**, detalhando a lógica matemática e os fluxos de dados mapeados.

Durante o desenvolvimento deste desafio, as maiores complexidades foram encontradas nas camadas de **Persistência** e **Negócio**, o que impossibilitou a conclusão total dessas frentes no prazo do treinamento.
